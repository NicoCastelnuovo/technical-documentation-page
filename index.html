Introduction

This is an introduction to Responsive Web Design, the practice to create responsive webpages, using the basic languages that developers use to build them:

  • HTML (Hypertext Markup Language) for content
  • CSS (Cascading Style Sheets) for design
You'll learn how to make webpages that respond to different screen sizes with CSS Flexbox, Grid and media-query.

HTML

HTML (Hypertext Markup Language) is a markup language that uses a special syntax or notation to describe the structure of a webpage to the browser. An HTML element is usually composed by:

  • A start tag
  • A content, like a text
  • An ending tag
Basically, the tags surround the referring content. There are a lot of tag, and each kind has a variety of attributes. There are also empty elements, which have no closing tag, like the <br> element (which has also no attributes) or the <img> element.

  <p>While 'h' in the last tags means 'heading','p' means 'paragraph'.</p>

CSS

CSS, or Cascading Style Sheets, tell the browser how to display the text and other content that you write in HTML. With CSS, you can control the color, font, size, spacing, and many other aspects of HTML elements. A CSS rule consist in:

  • The selector, which points to the HTML element you want to style
  • The declaration block, which contains properties and values

selector {
    property: value;
    property: value;
    property: value;
}

Here is an example of CSS code, that style the <h1> and <p> elements with the described parameters.

h1, p {
    color: blue;
    font-size: 16px;
    text-align: center;
}

Applied accessibility

In web development, accessibility refers to web content and a UI (user interface) that can be understood, navigated, and interacted with by a broad audience. This includes people with visual, auditory, mobility, or cognitive disabilities.

The following, is a code that you can use to visually hidden an element on the display, but make it accessible by the people which use a screen reader device.

.hidden {
    position: absolute;
    left: -10000px;
    width: 1px;
    height: 1px;
    top: auto;
    overflow: hidden;
}

Responsive Web Design Principles

There are many devices that can access the web, and they come in all shapes and sizes. Responsive web design is the practice of designing flexible websites that can respond to different screen sizes, orientations, and resolutions.The tools that you can use to make responsive a webpage are for example the CSS flexbox or the media query.

For example, the next code apply that body style only for the devices which have a viewport between 0 and 800px.

@media (max-width: 800px) {
  body {
    font-size: 21px;
    text-align: right;
    background-color: olive;
}
}
